Display 속성

DIsplay속성은 각각의 요소를 어떻게 보여줄지를 결정한다.

속성처리값? visibility(보여주기속성 - 기본값) : hidden(감추기) / inherit(부모요소의 값을 상속받는다.)

none : 보여주지 않을 때..
inline : 인라인상태(자신의 고유영역만 확보) - 단어의 의미로 해석?
block : 박스영역을 차지하는 속성 - 문단의 의미로 해석?
lnline-block : inline과 block의 합성
